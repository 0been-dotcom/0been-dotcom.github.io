---

블로그를 시작하고 싶었는데
플랫폼 선택부터 쉽지 않았다.

워드프레스는 매달 호스팅 비용이 발생하고,
네이버 블로그는 구글 애드센스 연결이 안 된다.

그러다 Jekyll이라는 걸 알게 됐다.

처음엔 개발자들만 쓰는 도구인 줄 알았는데,
막상 해보니 달랐다.

---

GitHub 계정을 만들고,
마음에 드는 테마를 Fork(내 계정으로 복사) 하면
그게 내 블로그가 된다.

Repository 이름을 `아이디.github.io` 로 설정하는 순간
바로 블로그 주소가 생긴다.

터미널도, 복잡한 코드도 필요 없었다.
GitHub 웹사이트 안에서 전부 해결됐다.

---

코딩을 몰라도 된다는 말이
처음엔 반신반의했는데,

직접 해보고 나서야 믿게 됐다.

---

앞으로 Jekyll을 처음 접하는 사람 입장에서
겪게 되는 것들을 하나씩 기록해볼 생각이다.
